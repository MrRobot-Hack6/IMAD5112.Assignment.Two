# Hack or Myth Quiz Game

## Myth or hack? You decide. This is a simple implementation of the IMAD instruction in Kotlin.

This is a simple implementation of the IMAD instruction in Kotlin. 

The app consists of three pages: a welcome page, a main page, and a result page. 

The main page contains four buttons: back, next, myth, and hack. 

The back and next buttons allow the user to navigate between the pages, while the myth and hack buttons allow the user to answer questions and see the results on the result page. 

The welcome page has a button to start the app, which navigates to the main page. 

The main page displays questions related to myths and hacks, and the user's score is updated based on their answers. 

The result page shows the final score and provides feedback based on the user's performance.
